# Free-quote-Developer-First API for Quote开发者的第一个行情API

简单好用易上手免费开源港股美股行情API

## 功能特性：

- 免费开源

- 获取港股实时行情

- 获取港股实时多档盘口

- 获取港股K线数据

- 获取美股实时行情

- 获取美股实时一档盘口

- 获取美股K线数据
